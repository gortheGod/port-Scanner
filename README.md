# port-Scanner
This is a simple port scanner of a target.

For termux:

pkg install python

pkg install git

git clone https://github.com/gortheGod/port-Scanner

cd port-Scanner

python port-Scanner.py / python3 port-Scanner.py


for linux:

apt-get install python3

apt install git

git clone https://github.com/gortheGod/port-Scanner

cd port-Scanner

python port-Scanner.py / python3 port-Scanner.py
